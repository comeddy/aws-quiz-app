# Step 8: Power 설치하기

> 원클릭 통합으로 Kiro를 확장합니다.

## 진행 순서

### 1. Powers 패널 열기

왼쪽 사이드바에서 **⚡ 번개 아이콘**을 클릭합니다.

### 2. 카탈로그 탐색

사용 가능한 Power를 스크롤하면서 각 Power의 기능과 활성화 키워드를 확인합니다.

### 3. "Install" 클릭

설치 완료! 설정 파일도, 터미널도 필요 없습니다. 바로 채팅에서 사용할 수 있습니다.

## 사용 가능한 Power 예시

| Power | 분야 | 설명 |
|-------|------|------|
| **Supabase** | 데이터베이스 | 데이터베이스, 인증, 스토리지 |
| **Netlify** | 배포 | 배포, 호스팅, 서버리스 |
| **Figma** | 디자인 | 디자인, UI, 컴포넌트 |
| **Postman** | API | API 테스팅, 컬렉션 |

> **💡 참고**
**시도해보세요**: Netlify Power를 설치한 후 Kiro에게 `"deploy my quiz app"`이라고 요청해보세요.
