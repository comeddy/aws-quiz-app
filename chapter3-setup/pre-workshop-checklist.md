# 사전 준비 체크리스트

> 실습을 시작하기 전에 아래 항목을 모두 완료했는지 확인하세요.

## 필수 준비 사항

### 1. Kiro 설치

[kiro.dev](https://kiro.dev)에서 Mac 또는 Windows용 설치 파일을 다운로드합니다. 워크샵 전에 설치하고 로그인까지 완료하세요.

### 2. Git 설치

Mac에서는 보통 사전 설치되어 있습니다. 터미널에서 확인:

```bash
git --version
```

### 3. Node.js 설치

[nodejs.org](https://nodejs.org)에서 LTS 버전을 다운로드합니다. MCP 서버(`npx`) 실행에 필요합니다.

### 4. Python + uv 설치

`uvx` 명령어 실행에 필요합니다.

```bash
# pip으로 설치
pip install uv

# 또는 Homebrew로 설치 (Mac)
brew install uv
```

> **⚠️ 주의**
**로그인 문제가 있나요?** GitHub, Google, 또는 AWS Builder ID로 로그인할 수 있습니다.
