# 인터페이스 둘러보기

> Kiro IDE의 주요 영역을 빠르게 살펴봅니다.

## Kiro IDE 주요 영역

### 1. Kiro Panel

왼쪽 사이드바에서 접근할 수 있으며, **Specs**, **Steering**, **Hooks** 관리를 담당합니다.

- **Specs**: 스펙 문서 생성 및 관리
- **Steering**: 프로젝트 컨텍스트 문서 관리
- **Hooks**: 이벤트 기반 자동화 설정

### 2. Powers

번개 아이콘(⚡)으로 접근하는 **원클릭 통합** 카탈로그입니다.

- 사용 가능한 Power 탐색
- 설치/제거 관리
- 키워드 기반 자동 제안

### 3. Chat

AI 에이전트와 대화하는 **채팅 패널**입니다.

- `⌘ + Shift + A` (Mac) / `Ctrl + Shift + A` (Windows)로 열기
- `#`을 입력하여 특정 파일을 컨텍스트로 첨부
- 자연어로 코드 변경 요청

### 4. Editor

코드를 작성하고 편집하는 **중앙 편집 영역**입니다. VS Code와 동일한 편집 경험을 제공합니다.
