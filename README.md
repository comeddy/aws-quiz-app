# Kiro 워크샵: AI와 함께 제로에서 빌딩까지

> **3시간 워크샵** — Spec-Driven Development, Steering, Hooks, Powers를 배우고 Kiro의 AI 에이전트로 실제 프로젝트를 구축합니다.

## 워크샵 소개

이 워크샵에서는 **Kiro IDE**를 사용하여 AI 기반 개발의 새로운 패러다임인 **Spec-Driven Development**(스펙 기반 개발)을 직접 체험합니다. 단순한 "프롬프트 코딩"을 넘어, 구조화된 계획과 실행으로 프로덕션 수준의 코드를 만드는 방법을 배웁니다.

## 대상

- AI 도구를 활용한 개발에 관심있는 개발자
- Kiro IDE를 처음 접하는 분
- Spec-Driven Development를 배우고 싶은 분

## 워크샵 진행 순서

| 순서 | 내용 | 시간 |
|------|------|------|
| 1 | Kiro 소개 | 30분 |
| 2 | 핵심 개념 (Specs, Steering, Hooks, MCP & Powers) | 30분 |
| 3 | 설치 & 인터페이스 둘러보기 | 30분 |
| 4 | 실습: AWS 퀴즈 앱 만들기 | 90분 |
| 5 | 베스트 프랙티스 & Q&A | 10분 |

## 사전 준비

워크샵을 시작하기 전에 다음 사항을 확인하세요:

- [x] Kiro IDE 설치 ([kiro.dev](https://kiro.dev))
- [x] Git 설치 (`git --version`으로 확인)
- [x] Node.js LTS 설치 ([nodejs.org](https://nodejs.org))
- [x] Python + uv 설치 (`pip install uv` 또는 `brew install uv`)
